# Rhythm

This is Sample Rhythm Game Project in Cocos2d-html5.

## Usage

### Download modules from github ###
    git clone git@github.com:oggata/Motorcycle.git

### run server ###
    python -m SimpleHTTPServer

### run game ###
    http://localhost:8000
